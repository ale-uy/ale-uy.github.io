Automation EDA

Clase escrita en python para automatizar con parametros preestablecidos, el analisis exploratorio de datos previamente cargados en pandas. Para usarlo solo tiene que poner el archivo "automation_eda.py" en la carpeta de su proyecto, importarlo "import automation_eda" e instanciar un objeto "mi_data_frame = EDA(df)" donde df es el dataframe que ya a creado con pandas
